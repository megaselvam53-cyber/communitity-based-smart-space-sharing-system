🚗 Smart Parking System (SpotConnect)

📌 Project Overview

SpotConnect is a simple and user-friendly smart parking web application.
It helps users find nearby parking spaces and book them easily.

Owners can also list their parking spaces and manage bookings.

---

💻 Technologies Used

- HTML
- CSS
- JavaScript

---

⚙️ Features

👤 User Module

- User Signup & Login
- Search nearby parking locations
- View parking details
- Book parking slots
- View booking history
- Rate & give feedback

🏠 Owner Module

- Owner Signup & Login
- Add parking spaces
- Manage availability
- View bookings
- Verify OTP & start/end parking

---

📂 Project Structure

- index.html → Home page
- user_dashboard.html → User main page
- owner_dashboard.html → Owner main page
- find_parking.html → Search parking
- booking pages → Booking flow
- app.js → Main JavaScript logic

---

⚠️ Current Limitations (Demo Version)

- OTP is demo-based (shown on screen), not real-time SMS OTP
- Data is stored using localStorage (no real database)
- Application works only on the same device/browser
- Multi-device access (e.g., two phones) is not supported currently
- Payment system is simulation only (no real transactions)

---

🚀 Future Enhancements

- Real-time database integration (MongoDB / Firebase)
- Real OTP system (SMS integration)
- Online payment gateway integration
- Multi-device support (access from multiple phones)
- Live GPS tracking
